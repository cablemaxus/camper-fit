# Camper Fit

Een kleine iPhone-vriendelijke PWA voor krachttraining onderweg met lichaamsgewicht en twee 5 kg dumbbells.

## GitHub Pages

Publiceer deze map vanuit de root van een GitHub repository:

1. Zet alle bestanden uit deze map in de root van de repository.
2. Ga in GitHub naar `Settings > Pages`.
3. Kies `Deploy from a branch`.
4. Kies branch `main` en folder `/root`.
5. Open de Pages URL zodra GitHub de site heeft gebouwd.

Daarna kun je de app op iPhone installeren via Safari: deelknop > `Zet op beginscherm`.
